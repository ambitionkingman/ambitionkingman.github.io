概念

Serialization（序列化）是一种将对象以一连串的字节描述的过程；反序列化 deserialization 是一种将这些字节重建成一个对象的过程。

java中，请求为 application/json时，参数为 json 字符串，封装成对象的过程为反序列化，Controller 中将对象转换成 json 字符串返回给前端的过程叫序列化。
